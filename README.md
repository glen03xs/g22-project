## User instructions

### To run on localhost

In the project directory, you can run:
#### git clone https://github.com/glen03xs/g22-project.git

To change to folder:
#### cd g22-project

To install the packages run: 
#### npm install

To run the application: 
#### npm run dev

The application run on  [localhost:6000](http://localhost:6000/).
